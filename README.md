# SwiftUI-YoutubePlayer

Simple project showing how to wrap a YTPlayerView into SwiftUI code. 

# Running
1. Install CocoaPods
2. Run `pod install` in the podfile directory from your terminal
3. Open up the `.xcworkspace` that CocoaPods created
